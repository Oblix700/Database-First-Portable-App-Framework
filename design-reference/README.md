# Design Reference

This folder is for visual and design examples that help AI tools understand the preferred app style.

Use this folder for:

- Sidebar screenshots.
- Table layouts.
- Detail popups.
- Drawers.
- Report builder examples.
- Calendar examples.
- Dashboard examples.
- Icons.
- Light and dark mode references.

## Design rule

Screenshots are references, not automatic requirements.

When using a screenshot, the AI must explain:

1. What design pattern is being reused.
2. Which part of the screenshot matters.
3. Which part should not be copied.
4. How it will fit the current app.
