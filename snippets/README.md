# Snippets

This folder will hold small reusable code patterns.

Snippets must be short, focused, and reusable.

Do not add large unfinished app files here.

Suggested future folders:

```text
snippets/react/
snippets/go/
snippets/sqlite/
snippets/echarts/
snippets/import-export/
snippets/report-builder/
```

## Snippet rules

Each snippet should explain:

1. What it does.
2. Where it should be used.
3. What dependencies it needs.
4. What should not be duplicated elsewhere.

## Reuse rule

Before creating a new snippet, check whether an existing one can be extended.
